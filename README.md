# EGUSPHERE-2025-4327

Datasets and analysis scripts associated with the article:

**Monsalve, A. and Link, O.**
*Grain roughness controls on velocity and bed stress fields around a fully protruding obstacle in supercritical flow*
EGUsphere, 2025
DOI: `10.5194/egusphere-2025-4327`

This repository provides the data summary, processed data, and scripts used for the calculations and analyses presented in the article.

## Important note on file truncation

Some files in this repository had to be truncated because of storage limitations. These truncated files are included to document the structure and content of the original datasets and to support transparency in the research workflow.

The data summaries and processed datasets included in this repository were generated from the original, non-truncated files and correspond to the data used in the article.

The complete, non-truncated files are available upon request. To request the full files, please contact:

**Angel Monsalve**
`amonsalve@uidaho.edu`

The complete files will be provided upon request.

## Repository contents

This repository includes:

* Data summaries used in the article
* Processed datasets generated from the original non-truncated files
* Analysis scripts used for calculations and post-processing
* Supporting files needed to reproduce the main analyses presented in the article

## Data availability and reproducibility

The processed data provided here correspond to the datasets used in the article. Because some large files were truncated before upload, users interested in fully reproducing the complete processing workflow from the original raw files should request the non-truncated files by email.

When using this repository, please cite both the associated article and the archived repository DOI once available through Zenodo.
